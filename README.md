# SelfDrivingCar-NN

## Description
Implementing Neural Networks from scratch in Machine Learning to build a Full Self-Driving (FSD) simulation. This project is a practical demonstration of how neural networks can be applied in the domain of self-driving vehicles.

## Features
- Neural Network implementation
- Simulation of Self-Driving car behaviors
- Real-time visualizations of sensors and decision-making processes

## Getting Started

### Prerequisites
- A browser that supports JavaScript

### Installation
1. Clone this repository
   ```bash
   git clone https://github.com/iamabhaydawar/SelfDrivingCar-NN.git
   ```
2. Navigate to the folder
   ```bash
   cd SelfDrivingCar-NN
   ```
3. Open `index.html` in a browser to start the simulation.

## Files
- `index.html` - Entry point of the simulation
- `style.css` - Styles for the simulation
- `car.js` - Manages car movement logic
- `controls.js` - Handles user and AI controls
- `main.js` - Executes the core application functionality
- `network.js` - Implements the neural network logic
- `road.js` - Handles road-related logic in the simulation
- `sensor.js` - Manages the car's sensor data
- `utils.js` - Provides utility functions
- `visualizer.js` - Visualizes the simulation and neural network processes
